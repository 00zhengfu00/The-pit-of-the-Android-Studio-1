  Learn about configuring HTTP proxies in Gradle

如果还是下载不了的话，请看下文介绍：



具体的android studio 配置


  安装步骤：
  1.先下载androidstudio，下载地址：http://developer.android.com/sdk/installing/studio.html 下载完直接安装，安装完打开它，然后会出现下载gradle-1.8-bin.zip，这个一般都会卡死，在天朝是这个样子的，不过不要紧，把androidstudio关掉。
  2.下载gradle-1.8-bin.zip，下载地址：http://www.gradle.org/downloads，下载完之后，找到目录： C:\Users\用户名\.gradle\wrapper\dists\gradle-1.8-bin\vruqmccc8532n7gr46qavsii8\ 把下载的gradle-1.8-bin.zip放到该目录下，并且把gradle-1.8-bin.zip解压到该目录(解压得到的文件为gradle-1.8)。
  3.最关键的来了，把该目录下的gradle-1.8-bin.zip.part文件删除，然后复制gradle-1.8-bin.zip.lck，把它改为gradle-1.8-bin.zip.ok
  4.ok，到此你就可以运行androidstudio开发了。
